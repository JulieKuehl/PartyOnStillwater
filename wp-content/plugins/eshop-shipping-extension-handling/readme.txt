Add Handling Fees for eShop Shipping Modules

== Description ==

eShop Shipping Extension framework overrides eShop's default shipping methods, interacting directly with third-party vendors for real-time shipping rates and services.

The Handling Fee Module enables the store owner to add Handling fees onto the prices returned by the shipping carriers.  

== Installation ==

1. Install eshop-shipping-extension plugin from Wordpress.org if you haven't already done so.;
1. Upload eshop-shipping-extension-handling.zip to your blog's wp-content/plugins directory;
1. Activate the plugin in your Plugin Admin interface;
1. Set eShop shipping preferences to Mode 4 (by Weight & Zone);
1. Go to Settings -> eShop Shipping Extension to activate and configure your Handling Fee Add-on.
